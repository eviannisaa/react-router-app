# Welcome to My App!

A React application built with React Router for a clean and structured development experience.

[See My Portfolio](https://evinannisa.github.io/eviannisa/)

## Tech Stack

- 🔒 TypeScript as the default for type safety
- 🎉 TailwindCSS for styling
- 📖 React Router for handling navigation and page routing within the React application ([see docs](https://reactrouter.com/))

## Getting Started

### Installation

Install the dependencies:

```bash
npm install
```

### Development

Start the development server with HMR:

```bash
npm run dev
```

Your application will be available at `http://localhost:5173/test-layout`.
